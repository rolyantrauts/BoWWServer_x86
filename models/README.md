
# Models

Trained from  [bcresnet](https://github.com/rolyantrauts/bcresnet) and https://github.com/rolyantrauts/wakeword_dataset/tree/main/gkws  
using just TTS and added noise with a limited training run.  
https://github.com/rolyantrauts/wakeword_dataset give a basic TTS of several engines to mix any recognisable TTS signatures.  
Using an ARPABET Sqlite dictionary to create unknown / adversarial words based on phonetic emphasis.
